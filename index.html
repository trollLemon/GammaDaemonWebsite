Gamma Daemon

Automatically Adjust Screen Brightness Based on Battery Life

GammaDaemon can operate within any Linux environment and is independent of init services. You can initiate it from various sources, such as an autorun script or an exec statement in a Window Manager config file.

GammaDaemon allows you to fine-tune screen brightness levels, ranging from 0 to 255, which can be customized to your preference. The configuration file is located at $HOME/.config/GammaDaemon/conf.toml. In the event that GammaDaemon is unable to locate this configuration file, it will default to using a built-in configuration.
